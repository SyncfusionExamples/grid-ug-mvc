# ASP.NET MVC DataGrid Getting Started Samples

This repository contains a collection of **ASP.NET MVC DataGrid sample projects** that demonstrate how to use and configure the DataGrid component in an MVC application. Each sample focuses on a specific DataGrid feature, helping developers understand practical implementation patterns through working examples.

## Repository description

The project showcases multiple DataGrid scenarios implemented using **ASP.NET MVC**. The attached samples illustrate common requirements such as data binding, column customization, responsiveness, and user interaction features. These examples are organized as individual MVC projects, making it easy to explore and run each scenario independently.

## Included Samples

The repository includes MVC projects demonstrating the following DataGrid concepts:

- Getting started with ASP.NET MVC DataGrid
- Basic and complex data binding
- Auto-generated and auto-fit columns
- Column resizing, reordering, and chooser
- Column menu and header customization
- Template-based columns
- Responsive column behavior
- Order details and structured data presentation

Each folder represents a focused MVC sample showcasing a specific DataGrid capability.

## Prerequisites

To run the samples locally, ensure the following tools are available:

- Visual Studio 2022
- ASP.NET MVC development workload
- NuGet package restore enabled

## How to Run the Samples

1. Clone or download this repository to your local machine.
2. Open the required sample solution in **Visual Studio 2022**.
3. Restore NuGet packages by rebuilding the solution.
4. Run the project to view the DataGrid output in the browser.

## Usage

These samples are intended for learning and reference purposes. Developers can use them to understand how different DataGrid features behave in an MVC environment and adapt the implementations for real-world applications.
